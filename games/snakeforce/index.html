Snake Force

Game Description

Snake Force is an exciting arcade game where you control a snake through various challenges. Collect power-ups, avoid obstacles, and try to achieve the highest score!

Controls

  • ← → Move left/right
  • ↑ ↓ Move up/down
  • Space Special ability
